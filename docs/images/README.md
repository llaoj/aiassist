## 🎨 制作建议

### 动画制作工具推荐：
1. **Asciinema** - 录制终端会话
   ```bash
   asciinema rec demo.cast
   # 转换为 GIF
   agg demo.cast demo.gif
   ```

2. **Terminalizer** - 更炫酷的终端录制
   ```bash
   terminalizer record demo
   terminalizer render demo
   ```

3. **VHS** (charm.sh) - 代码方式生成终端动画
   ```vhs
   Output demo.gif
   Type "aiassist"
   Enter
   Sleep 1s
   Type "服务器负载很高"
   Enter
   Sleep 2s
   ```

### 静态图制作工具：
- **Excalidraw** - 手绘风格，适合架构图
- **draw.io** - 专业流程图
- **Figma** - 专业设计
- **Carbon** - 代码截图美化

### 尺寸建议：
- GIF动画：800x600 或 1000x750
- 静态图：1200x800 或更大
- Logo：已有 logo.svg

### 文件命名规范：
- 使用英文小写+连字符
- GIF 动画：xxx.gif
- 静态图：xxx.png
- 所有文件放在 `docs/images/` 目录
