# aiassist
AI Shell Assistant (aiassist): an intelligent CLI tool for server &amp; cloud-native O&amp;M. It provides expert-level diagnosis, solutions and command guidance for O&amp;M engineers, optimizing workflows, cutting fault troubleshooting time and boosting O&amp;M efficiency. Works on bash/sh/zsh, installable via yum/apt/apk, out-of-the-box.
